INTRODUCTION¶

In recent years, we have seen that machine learning has grown as an emerging technology in the world. It has its implementation almost everywhere such as Image Recognition, Speech Recognition, Traffic Recognition, Self-driving cars, and many more. ,. One area where machine learning can be applied is in predicting whether a patient is suffering from heart diseases or not. This information help patient to better understand risk in heart diseases. This prediction can reduce the risk of heart attack.

In this project, we will be using a data set to predict whether a heart attack may or may not occur to patient. For this we have use SVM machine learning algorithm to build a predictive model. Machine learning algorithm to build a predictive model.

Our goal is to build a predictive model that accurately identifies if the patient suffering from heart attack or not based on the factor associated such as their age, sex, cholesterol etc. By leveraging machine learning techniques, we can improve the accuracy and efficiency of these processes, and ultimately provide better service to our patient.

PROBLEM STATEMENT¶

The problem statement for this machine learning project is to predict whether a patient is suffering from heart attack or not cause Heart attack is the one of the major concern, people are facing nowadays and its not limited to any certain age group .This project aims to detect the attack at the early stage with the help of SVM machine learning algorithm with great efficiency and accuracy.

In order to solve the problem, we are developing a machine learning model that will allow users to anticipate whether or not they will experience a heart attack.

In [20]:
def mymodel(model):
    model.fit(x_train,y_train)
    y_pred = model.predict(x_test)
    print('Accuracy score :',accuracy_score(y_test,y_pred))
    print(classification_report(y_test,y_pred))
    
    train = model.score(x_train,y_train)
    test = model.score(x_test,y_test)
    
    print('training performance :',train)
    print('testing performance :',test)
    
    return model
